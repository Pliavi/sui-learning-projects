# Sui Jokenpo

Um jogo simples de Jokenpo (Pedra, Papel, Tesoura) implementado na blockchain Sui usando a linguagem Move para o Bootcamp da Sui Brasil

# O que foi aprendido

- Como utilizar options
- Como definir enums
- Instancias de enums não funcionam fora dos módulos que os definem, é necessário criar funções públicas para instanciá-los
